# Security Policy

## Reporting a Vulnerability

If you believe you have found a security vulnerability in airgeddon, please report it privately instead of opening a public issue.

You can contact the airgeddon staff through any of the methods listed in the Contact section of the wiki:

https://github.com/v1s1t0r1sh3r3/airgeddon/wiki/Contact

Please include as much detail as possible so the issue can be reviewed and reproduced.

## Security Advisories

If the vulnerability is confirmed, the maintainers may use GitHub Security Advisories to coordinate the fix and publish information when appropriate.

## Supported Versions

Security fixes are only provided for the latest stable release available from the master branch. Development, beta, testing, legacy, or otherwise modified versions are not covered.
